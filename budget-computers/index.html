About Us

At PNW Budget Computers, our goal is to give desktop and laptop computers a second chance at life, helping to reduce e-waste.
We also strive to make computing accessible to members of the local community w-ho are in need of an affordable system that is capable of performing everyday computing tasks.

We focus on rebuilding systems that are ideal for:

  • Web Browsing
  • Email
  • Home Office
  • School
  • Media Streaming
  • Light to Medium PC Gaming

We upgrade and refurbish systems to ensure that they're the best that they can be.

  • We replace slow old magnetic hard disk drives (HDDs) with faster solid-state drives (SSDs)
  • We install larger and faster RAM modules so programs perform better and have more memory to work with
  • We install the latest supported up-to-date version of Windows with a custom configuration applied to remove bloat and unnecessary features in order to improve performance.
    If a system is not supported by Windows, we install a Linux distribution that provides a familiar, lightweight, and fully-featured desktop environment.
